1. Giải nén file ZIP.
2. Copy toàn bộ file bên trong vào đúng thư mục GitHub Desktop đang quản lý.
3. Chọn Replace all / Ghi đè tất cả.
4. GitHub Desktop: Commit to main, sau đó Push origin.
5. Chờ Cloudflare Pages 1-3 phút.
6. Mở https://thanhbinhbds.id.vn/admin/?v=20260712 và nhấn Ctrl+F5.
