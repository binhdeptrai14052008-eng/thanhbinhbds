# Thanh Bình BĐS V3

## Cài đặt

1. Giải nén bộ mã.
2. Copy toàn bộ file vào repo `thanhbinhbds`.
3. Commit và Push lên GitHub.
4. Chờ Cloudflare Pages deploy.

## Link

- Trang chủ: `https://thanhbinhbds.id.vn/`
- Quản trị: `https://thanhbinhbds.id.vn/admin/`

## Facebook và TikTok

Mở file:

`js/firebase-config.js`

Thay:

```js
facebookUrl: "https://www.facebook.com/",
tiktokUrl: "https://www.tiktok.com/"
```

bằng link Facebook và TikTok thật của bạn.
